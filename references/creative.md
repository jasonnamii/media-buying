# 크리에이티브 설계·스펙 (훅·다양성·플랫폼별 최적화)

## 1. 크리에이티브가 최대 레버인 이유

**성과 변동성**:
- CTR 변동: 56% (광고 간 차이)
- CPA 변동: 44% (좋은 크리에이티브는 50% 저감)
- ROAS 영향도: 크리에이티브 > 오디언스 > 입찰

**결론**: 타겟팅·입찰 최적화보다 **크리에이티브 다양성**이 가장 빠른 성과 향상

## 2. 3초 훅 법칙 & Hook Rate

### 2.1 Hook Rate 정의 & 벤치마크

```
Hook Rate = 3초 재생 이상 / 총 임프레션

건강도:
├─ 25~30% = 정상 범위 (기본 운영 가능)
├─ 40%+ = "유니콘" (안정적 스케일링 가능)
└─ <25% = 위험 (알고리즘이 저품질로 판정)
```

**영향도**:
- Hook rate 25% → CPM 기본값
- Hook rate < 15% → CPM 2배, 노출 30% 감소
- Hook rate 40% → CPM 30% 절감, 노출 50% 증가

### 2.2 Pattern Interrupt 전략

```
사용자 스크롤 흐름:
┌─────────────────────────┐
│ 광고 노출 (1초)         │
│ → 스크롤? 또는 관심?    │
│ (1.7초 평균 결정 시점) │
└─────────────────────────┘

Pattern Interrupt 적용:
1초: 강한 색상 변화 / 동작 / 질문
2초: 페인포인트 명확화
3초: 제품·브랜드 노출
```

### 2.3 Hook 요소 체크리스트

- [ ] 처음 1초: 색상 대비 또는 동작 변화
- [ ] 처음 3초: 페인포인트 해결 명시
- [ ] 사람 얼굴 또는 감정 표현? (주목도 +40%)
- [ ] 텍스트 크기 충분? (모바일 읽기 가능)
- [ ] 동영상: 30fps 이상? (부드러움)

## 3. ABCD 프레임 (Google Attention·Branding·Connection·Direction)

### 3.1 4단계 구조

| 단계 | 시간 | 목표 | 사례 |
|------|------|------|------|
| **A: Attention** | 0~1초 | 주목 집중 | 동작·색상 변화 |
| **B: Branding** | 1~3초 | 브랜드 인식 | 로고·색상·제품 |
| **C: Connection** | 3~10초 | 감정 공감 | 페인포인트·욕망 |
| **D: Direction** | 10초+ | 행동 유도 | CTA·클릭 명확 |

### 3.2 효과

```
ABCD 적용:
├─ 단기 판매: 30% 상승
└─ 장기 브랜드: 17% 상승
```

## 4. Creative Diversity Principle (Meta)

### 4.1 진정한 다양성

```
❌ 수량 다양성 (무의미):
   같은 제품을 30개 각도로 촬영
   → Meta 알고리즘이 중복으로 판정

✓ 질적 다양성 (유효):
   ├─ 스타일: UGC vs 브랜드필름 vs AI
   ├─ 톤: 유머 vs 심각 vs 감정적
   ├─ 포맷: 영상 vs 이미지 vs 카로셀
   └─ 메시지: 가격 vs 품질 vs 신뢰도
```

### 4.2 Diversity Score (Meta)

```
Meta Ads Manager → Creative Insights → Diversity Score

목표: 6~8/10 (양호)
<5/10: 로테이션 급진행 필요
```

## 5. UGC vs. 브랜드필름 효과성 비교

### 5.1 핵심 지표 (2026)

| 지표 | UGC | 브랜드 | 비율 |
|------|-----|--------|------|
| **CTR** | 4.0% | 1.0% | UGC 4배 우수 |
| **CPC** | $0.30 | $0.60 | UGC 50% 저감 |
| **구매 전환 가능성** | 97% | 50% | UGC 94% 높음 |
| **신뢰도** | 79% | 52% | UGC 신뢰도 우수 |
| **수익/방문자** | $154 | $100 | UGC 54% 높음 |

### 5.2 플랫폼별 성과

| 플랫폼 | UGC 우위 | 주의사항 |
|--------|--------|---------|
| **TikTok** | 22% (가장 높음) | 네이티브 톤 필수 |
| **Meta** | 15~18% | 브랜드 신뢰 병행 |
| **YouTube** | 10배 (수직) | 수평 영상 피함 |

### 5.3 UGC 제작 가이드

```
효과적인 UGC:
├─ 실제 사용자 (연기 느낌 제거)
├─ 한국어 자막·음성 오버
├─ 일상 배경 (스튜디오 피함)
├─ 페인포인트 먼저 노출
└─ 솔루션 후 만족감 표현

비용: 영상당 $100~500 (스튜디오 $5,000 vs 1/10)
```

## 6. 플랫폼별 크리에이티브 스펙

### 6.1 Meta (Facebook·Instagram)

**2026 최적 스펙:**

| 유형 | 해상도 | 비율 | 권장사항 |
|------|-------|------|---------|
| Reels/Stories | 1080x1920 | 9:16 | 필수 (100% 화면) |
| 인피드 이미지 | 1080x1350 | 4:5 | CTR 1% 상승 |
| 카로셀 | 1200x628 | 1.91:1 | 각 카드별 텍스트 |

**텍스트 오버레이 제한:**
- Meta 정책: 20% 초과 금지 (이미지)
- Tool: Facebook Image Text Check

### 6.2 TikTok

**필수 스펙:**
```
비율: 9:16 (풀 스크린)
해상도: Full-HD (1080x1920+)
최소: 540x960px (모바일 전체 점유)

주의: 정사각형/가로 = 빈 공간 → 참여도 -30%
```

### 6.3 YouTube

**성능 데이터:**
```
수직 영상 > 수평 영상: 90% 완청률 우수

Bumper (6초):
├─ 스킵 불가
└─ 목표: Branding (메모리)

TrueView (6~30초):
├─ 처음 5초가 결정적
└─ 목표: Engagement

Skippable (30초+):
├─ 후반부 전환 유도
└─ 목표: 완청률 높이기
```

### 6.4 검색 플랫폼 (Naver·Google)

| 플랫폼 | 해상도 | 스펙 |
|--------|-------|------|
| **Google GFA** | 300x250·728x90·반응형 | 가로 비율 선호 |
| **Naver 비즈보드** | 1029x258 | 이미지 최상단 배치 |
| **Kakao 비즈보드** | 1029x258 | 네이티브 톤 |

### 6.5 LinkedIn (B2B)

```
권장: 1200x627px (16:9)
영상: 15~30초 (자막 필수)
포맷: 해시태그·매력적 제목 (CTR+25%)
```

## 7. 자산 개수 가이드

### 7.1 Meta (Meta/Instagram)

```
권장: 4~6 광고 / ad set

구성:
├─ UGC 2개 (상이한 페인포인트)
├─ 브랜드필름 1개
├─ 카로셀 1개
└─ 이미지 1개 (옵션)

효과: 개수 ↑ = Fatigue 회피 = ROAS 유지
```

### 7.2 TikTok

```
권장: 3~5개 (신규로테이션 2주 단위)
```

### 7.3 Google Performance Max (Asset Group)

```
필수 자산 (모두 제공):
├─ Headline: 최소 15개 (필수)
├─ Description: 최소 5개
├─ 이미지: 최소 1개 (4개 이상 권장)
└─ 영상: 최소 1개 (권장)

효과: 완전성 → 자동 최적화 성능 +30%
```

## 8. 캡션·자막·텍스트 오버레이

### 8.1 자막 필수 여부

| 플랫폼 | 자막 | 음량 설정 | 주의사항 |
|--------|------|---------|---------|
| **TikTok** | 필수 (95%) | Mute 기본 | 영어·한국어 병기 |
| **Meta** | 선택 (50%) | Mute OK | 텍스트 20% 제한 |
| **YouTube** | 선택 (70%) | 음성 기본 | 자막 = +25% 완청 |

### 8.2 텍스트 오버레이 배치

```
효과적 배치:
1초: 훅 텍스트 (작음, 가독성)
   "3초 만에 피부 변화"
3초: 브랜드명 (상단 중앙)
10초: CTA (하단 우측)
   "지금 보기" / "구매하기"
```

### 8.3 한국 텍스트 가이드

```
피해야 할 표현:
├─ "~하는군요" (뻣뻣함)
├─ 과도한 이모지 (저품질 느낌)
├─ 모두 대문자 (거슬림)

권장:
├─ 1인칭 "저도", "우리도"
├─ 친근한 반말 (톤에 따라)
└─ 한정사 사용 ("이제 가능")
```

## 9. 제품·인물·가격·CTA 배치

### 9.1 황금 배치 규칙

```
비디오 영상:
0~1초: 인물 클로즈업 (눈 마주침)
1~3초: 페인포인트 (감정 표현)
3~5초: 제품 소개 (명확한 노출)
5~8초: 기능·가치 (해결점)
8~10초: 가격 (선택사항)
10초+: CTA (명확한 버튼·텍스트)
```

### 9.2 CTA 버튼 배치

```
권장: 하단 우측 또는 중앙
크기: 터치 가능 (최소 48px × 48px)
색상: 배경과 대비 높음

효과: 잘못된 배치 = CTR -20%
```

### 10.1 성능 벤치마크 (2026)

| 지표 | AI | 수동 제작 | 트레이드오프 |
|------|-----|---------|-------------|

| **저가 제품 전환** | 동등 | 기준 | 가격 이점 |
| **고가 제품 전환** | -8~14% | 기준 | 브랜드 신뢰 손실 |

### 10.2 AI 활용 전략

```
권장:
├─ $100 이하 제품 → AI 50%
├─ $100~500 → AI 20% + 수동 80%
└─ $500+ → 수동 100% (브랜드 보호)

Meta Symphony (AI 생성):
└─ CTR 12% 높음, 단 고가 전환 약함
```

## 11. 크리에이티브 로테이션 운영

### 11.1 생명주기 관리

```
Week 1~2: 신규 크리에이티브 런칭 (학습 페이즈)
Week 2~4: 성과 모니터링 (Hook rate 25%+ 확인)
Week 4~8: High Performer 유지 (예산 집중)
Week 8+: Fatigue 신호 감시 (Frequency 3.5+ ?)

Fatigue 신호:
├─ CTR 20% 급락
├─ Frequency 4.0회/주 이상
└─ Hook rate 하강 중
  → 즉시 신규 크리에이티브 3~5개 추가
```

### 11.2 Evergreen vs. Seasonal

| 유형 | 로테이션 | 주기 | 예시 |
|------|---------|------|------|
| **Evergreen** | 월 1회 신규 | 3~6개월 | 제품 설명 |
| **Seasonal** | 주 1회 신규 | 4~8주 | 시즈널 오퍼 |
| **Trending** | 거의 매일 | 1~2주 | 바이럴 요소 |

## 12. 크리에이티브 브리프 템플릿

### 12.1 기본 구조

```
프로젝트: [캠페인명]
대상: [오디언스·특성]
주요 페인포인트: 
  1. [사용자 관점 문제]
  2. [비용·시간 관점]

핵심 메시지:
  "우리 제품은 [기능] + [감정]"

CTA:
  - 주 CTA: "구매하기" / "지금 신청"
  - 보조 CTA: "더 알아보기"

톤:
  □ 유머 □ 진지 □ 감정적 □ 교육적

금지어:
  - 경쟁사 명칭
  - 과장 표현
  - 법적 이슈

참고 사항:
  - 배경: 흰색 권장
  - 음악: 저작권 자유곡 (YouTube Audio Library)
  - 길이: TikTok 15~30초, Meta 15초
```
